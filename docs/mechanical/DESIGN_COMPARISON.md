## 4. 幾何学計算

### 120°扇形のパラメータ（v4.1）

```
前面弦長 = 440mm
弦 = 2R × sin(θ/2) = 2R × sin(60°) = 1.732R
R (前面半径) = 440 ÷ 1.732 = 254.0mm

奥行き = 243mm
r (背面半径) = 254 - 243 = 11mm
背面幅 = 2 × 11 × sin(60°) = 19mm

扇形面積 = (θ/2)(R² - r²)
         = (π/3)(254² - 11²)
         = (π/3)(64516 - 121)
         = 66,825 mm²

全体容積 = 66,825 × 232 = 15.5L
```

### 楕円のパラメータ（正面図）

```
長軸 2a = 440mm → a = 220mm
短軸 2b = 232mm → b = 116mm

楕円方程式: (x/220)² + (y/116)² = 1
```

### ドライバー収まり検証

```
ドライバー位置: X = ±130mm (中心から)
ドライバー直径: Ø68mm (半径34mm)
ドライバー上下間隔: 70mm (中心間)

X = 130mm での楕円高さ:
h = 2 × 116 × √(1 - (130/220)²)
h = 232 × √(1 - 0.349)
h = 232 × 0.807
h = 187.2mm

楕円範囲: ±93.6mm (中心から)
ドライバー外縁: ±69mm (中心から)

結果: 69mm < 93.6mm → ドライバーは楕円内に収まる ✓
```

## 5. Gemini考案デザイン vs B&W Traceデザイン

### Pros & Cons 比較

| 観点 | Gemini v3.5 | B&W Trace v4.1 |
|------|-------------|----------------|
| **サイズ** | ✅ コンパクト (380×220×200) | ❌ やや大きい (440×243×232) |
| **Formation Wedge忠実度** | ❌ 低い（オリジナル解釈） | ✅ 高い（公式寸法準拠） |
| **ドライバー収まり** | ❌ はみ出し問題あり | ✅ 楕円内に完全収納 |
| **背面幅** | 30mm（鋭すぎる） | 19mm（より鋭い、B&W的） |
| **製造難易度** | ⚠️ 極端なテーパーで困難 | ⚠️ 同様に難しいが計算済み |
| **音響容積** | ~9L | ~15.5L（より余裕あり） |
| **美的バランス** | ❌ 寸法が不釣り合い | ✅ B&W設計の黄金比 |

### 詳細分析

#### Gemini v3.5 の長所
1. **コンパクト設計**: 小型で設置場所を選ばない
2. **軽量**: 容積が小さいため軽くなる可能性
3. **独自性**: オリジナル解釈によるユニークなデザイン

#### Gemini v3.5 の短所
1. **ドライバー収まり問題**: 極端なテーパーでスピーカーがはみ出す
2. **Formation Wedgeらしさの欠如**: 公式寸法と大きく異なる
3. **音響的制約**: 容積不足でスピーカー性能を発揮できない可能性
4. **製造困難**: 30mm背面は組立・シール困難

#### B&W Trace v4.1 の長所
1. **公式寸法準拠**: Formation Wedgeの外観を正確に再現
2. **ドライバー完全収納**: 数学的に検証済み
3. **十分な音響容積**: 15.5Lでスピーカー性能を発揮
4. **設計の整合性**: 120°という角度がB&W公式と一致

#### B&W Trace v4.1 の短所
1. **サイズ増加**: 20%程度大きくなる
2. **重量増加**: 容積増加に伴う重量増
3. **設置スペース**: より広い設置面積が必要

---

## 6. 推奨事項

### 結論: v4.1 (B&W Trace) を推奨

**理由:**
1. Formation Wedgeのインスパイアを目的とするなら、公式寸法に準拠すべき
2. ドライバー収まり問題が解決されている
3. 音響的に十分な容積を確保
4. 120°という角度がB&W公式と一致（設計意図を尊重）

### 妥協案（必要な場合）

サイズを抑えたい場合、以下の調整が可能:
- 幅を420mmに戻す → ドライバーX位置を115mmに調整
- 高さを200mmに戻す → ドライバーY間隔を60mmに縮小
- ただし、これはFormation Wedgeとの乖離を意味する
